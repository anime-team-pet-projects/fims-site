# Films Backend

# Установка
- Скопируйте проект
    ```console
    $ git clone https://github.com/anime-team-pet-projects/fims-site.git -b backend
    ```
- Скопируйте `.env`, либо создайте руками
    ```console
    $ cp .env.example .env
    ```
- Установите параметры в файле `.env`
- Запустите контейнер
    ```console
    $ docker-compose up -d --build
    ```
